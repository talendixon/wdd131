SkillTracker Site Plan

WDD 131 Personal Project

Site Name

SkillTracker

Purpose

SkillTracker is designed to help users track their progress in developing new skills. The application allows users to log practice sessions, set goals, visualize progress over time, and receive customized recommendations based on their learning patterns. Its purpose is to motivate consistent practice and provide data-driven insights for more effective skill development.

Audience

The target audience includes adult learners (18-45) who are self-motivated to develop new skills outside of formal education settings. This includes professionals looking to expand their capabilities, hobbyists tracking their creative pursuits, and individuals learning new languages or instruments. The audience is moderately tech-savvy and values organization and data visualization to support their learning journey.

Dynamic Elements

JavaScript will be used to create the following dynamic elements:

1. Interactive Skill Tracker Dashboard

  • Skill progress visualization using chart.js
  • DOM manipulation to update progress bars and statistics in real-time
  • Event listeners for form submissions when logging practice sessions

2. Goal Setting and Achievement System

  • Object-oriented approach to store and track user goals
  • Conditional branching to deliver different messages based on goal completion status
  • Local storage utilization to persist user data between sessions

3. Practice Timer with Analytics

  • Array methods (map, filter, reduce) to analyze practice session data
  • ES Modules to separate timer functionality from data processing
  • DOM interaction to start/pause/stop timer and display results

4. Recommendation Engine

  • Array filtering to identify practice patterns
  • Conditional logic to generate personalized recommendations
  • DOM manipulation to display contextual suggestions
